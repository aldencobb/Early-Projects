CSS Display and Positioning Reference Table

Tag

Name

Description
position: static The element is positioned according to the normal flow of the document; default value.
relative The element is positioned according to the normal flow of the document and offset relative to itself.
absolute The element is removed from normal document flow and positioned relative to its closest ancestor.
fixed The element is removed from normal document flow and positioned relative to the nearest containing block.
sticky The element is positioned according to the normal flow of the document and offset according to its nearest scrolling ancestor and containing block (nearest block-level ancestor).
